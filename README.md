# Advanced Python CLI Calculator

A command-line calculator built using **Python** that supports both basic arithmetic operations and scientific functions.
This project demonstrates core Python concepts such as loops, condition handling, file handling, and mathematical operations.

---

## 🚀 Features

* Basic arithmetic operations

  * Addition
  * Subtraction
  * Multiplication
  * Division

* Scientific functions

  * Square Root
  * sin(x)
  * cos(x)
  * log(x)
  * factorial

* Expression evaluation

* Calculation history tracking

* Saves history to a file (`history.txt`)

* Error handling for invalid input

---

Python loops (while)

Functions
